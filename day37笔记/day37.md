### HTML

~~~
html常见的标签？
	1.结构体标签
	2.文本标签
	3.图片标签
	4.列表标签
	5.特殊符号
	6.块级标签	
	7.语义化标签
	8.超链接标签   
	9.表格标签
	10.表单标签
		表单标签：<form></form>
			注意：表单是一个看不到摸不到的东西
			作用：可以收集用户输入的各种数据，可以和服务器进行数据的交互
			
			常见的属性？
				action属性：表单提交到哪个路径
				method属性：表单提交方式
					get：默认值，表单项的数据会展示在浏览器的地址栏上
					post：表单项的数据不会展示在浏览器的地址栏上
					
					推荐使用post，因为安全
			
		常见的表单项？
			1.文本框
				<input/>
				
				常见的属性？
					type属性：设置文本框的类型
						type="text" 普通文本框
						type="password" 密码框
						type="radio" 单选框
						type="date" 日期框
						type="checkbox" 多选框
						type="email" 邮箱框(自带邮箱校验功能)
						type="file" 文件框
						type="color" 颜色框
						type="number" 数字框
						type="hidden" 隐藏框
						type="button" 普通按钮框
						type="submit" 按钮(自带提交功能)
						type="reset" 按钮(自带重置功能)
						
					checked属性：
						设置哪个被选中，只能用在单选框和多选框上面
						
					placeholder属性：
						设置文本框提示内容
						
			2.下拉框标签
				<select></select>
				<option></option>
				
				常见的属性？
					selected属性：设置哪个option默认被选中
					
			3.文本域
				<textarea></textarea>
				
			4.label标签
				<label for="username">用户名：</label>
				<input type="text" id="username">
				
				注意：label标签中的for属性要和关联标签的id属性值相同
			
			标签项标签的通用属性？
				id属性：代表标签的身份证号，id属性值都是唯一的，唯一标识
				name属性：代表标签的名字，未来我们会在服务器端根据name属性值来获取表单项数据
				value属性：代表标签的输入框的数据，未来我们会在服务器端根据name属性值来获取value属性值
~~~

### CSS

~~~
什么是CSS?
	层叠样式表（cascading style sheet），也叫做级联样式表
	
	层叠？
		是指多个样式可以施加在同一个标签上
	
	样式？
		设置颜色，大小，宽高等等都属于样式设置
		
css的作用？
	1.美化页面的
	2.可以使页面中的结构体代码和样式代码进行分离
		a.提高了代码的复用性
		b.提高了代码的维护性
		c.提高了代码的开发效率
	
css的引入方式？
	方式一：在html的标签中添加style属性，在style属性中写样式
		格式：<div style="属性名:属性值; 属性名:属性值; ......">约吗</div>
		
		当我们只需要给某一个标签施加一些样式的时候，就可以选择使用方式一
		
	方式二：在head标签中写style标签，在style标签中写选择器
		格式：
			<head>
				<style>
					选择器 {
						属性名:属性值;
						属性名:属性值;
					}
					
					选择器 {
						属性名:属性值;
						属性名:属性值;
					}
				</style>
			</head>
			
		当一个页面中的多个标签施加相同的样式的时候，可以选择使用方式二
	
	方式三：在head标签中写link标签，通过link标签引入外来的css文件
		格式：
			<head>
				<link rel="stylesheet" href="css文件路径"/>
			</head>
			
			css文件：
				选择器 {
					属性名:属性值;
					属性名:属性值;
                }

                选择器 {
                	属性名:属性值;
                	属性名:属性值;
                }
	
		当多个页面中的多个标签施加相同的样式的时候，可以选择使用方式三

	注意：如果三种方式施加在同一个标签上的话，样式展示是就近原则

css的选择器？
	什么是选择器？
		能够帮助我们快速定位到某一个或者某几个标签的东西，就称为选择器
		
	css常见的选择器？
		类型选择器(标签选择器)？
			格式：
				标签名 {
					属性名:属性值;
					属性名:属性值;
					... ...
				}
			
		ID选择器？
			格式：
				#id属性值 {
					属性名:属性值;
					属性名:属性值;
					... ...
				}
		
		类选择器？
			格式：
				.class属性值 {
					属性名:属性值;
					属性名:属性值;
					... ...
				}
		
		通配符选择器？
			格式：
				* {
					属性名:属性值;
					属性名:属性值;
					... ...
				}

		包含选择器(后代选择器)？
			格式：
				标签名 标签名 {
					属性名:属性值;
					属性名:属性值;
					... ...
				}
	
		子元素选择器？
			格式：
				标签名 > 标签名 {
					属性名:属性值;
					属性名:属性值;
					... ...
				}
				
		组合选择器？
			格式：
				选择器, 选择器，... {
					属性名:属性值;
					属性名:属性值;
					... ...
				}

		属性选择器？
			格式一：
				标签名[属性名] {
					属性名:属性值;
					属性名:属性值;
					... ...
				}
				
			格式二：
				标签名[属性名 = 属性值] {
					属性名:属性值;
					属性名:属性值;
					... ...
				}
				
		复合选择器？
			<style>
                /* b > p {
                    color: red;
                } */

                /* #b2 > p {
                    color: red;
                } */

                /* .aaa > p {
                    color:red;
                } */

                /* #b3 > .ccc {
                    color: red;
                } */

                #b3 > p[class] {
                    color: red;
                }
            </style>
            <body>
                <b id="b1" class="aaa">
                    <p>aaa</p>
                    <div>
                        bbb
                        <p>ccc</p>
                    </div>
                </b>
                <b id="b2" class="bbb">
                    <p>aaa</p>
                    <div>
                        bbb
                        <p>ccc</p>
                    </div>
                </b>
                <b id="b3" class="aaa">
                    <p class="ccc">aaa</p>
                    <p class="ddd">aaa</p>
                    <div>
                        bbb
                        <p>ccc</p>
                    </div>
                </b>
            </body>

		伪类选择器？
			:link 设置 a 对象在未被访问前的样式
			:hover 设置对象在其鼠标悬停时的样式
			:active 设置对象在被用户激活(在鼠标点击与释放之间发生的事件)时的样式
			:visited 设置 a 对象在其链接地址已被访问过时的样式
			以上几种伪类选择器主要是针对超链接标签
			
			:focus 设置对象在成为输入焦点(该对象的 onfocus 事件发生)时的样式
