### 网络编程

~~~
什么是网络编程？
	是指通过我们写好的程序，可以实现数据在互联网端进行传输
	网络编程也叫作套接字编程，也叫做Socket编程
	
网络编程三兄弟(三要素)？
	1.IP地址
	2.端口号
	3.协议
	
IP地址？
	IP地址代表了计算机的唯一标识
	
	我们自己的IP怎么查看？
		ipconfig
		ipconfig -all

	我们可以通过IP地址来判断网络是否连通？
		ping  IP地址
		
端口号：
	端口号代表计算机中的应用程序的唯一标识
	
	所谓的端口，就好像是门牌号一样，客户端可以通过ip地址找到对应的服务器端，但是服务器端是有很多端口的，每个应用程序对应一个端口号，通过类似门牌号的端口号，客户端才能真正的访问到该服务器。为了对端口进行区分，将每个端口进行了编号，这就是端口号
	
	端口号只有整数，范围是从0 到65535
	
	如果端口号被占用怎么办？
		1.换一个端口号即可
		2.把占用端口号的应用程序找到，关闭即可
			怎么找到是谁占用了这个端口号呢？
				netstat -ano 查看应用程序的PID(进程ID)是多少
	
协议：
	协议，网络协议的简称，网络协议是通信计算机双方必须共同遵从的一组约定。如怎么样建立连接、怎么样互相识别等。只有遵守这个约定，计算机之间才能相互通信交流。
	为了使数据在网络上从源到达目的，网络通信的参与方必须遵循相同的规则，这套规则称为协议（protocol），它最终体现为在网络上传输的数据包的格式
	
	UDP协议
	*TCP协议
	网络通信协议
~~~

### InetAddress

~~~
什么是InetAddress类？
	此类表示互联网协议 (IP) 地址

常见的成员方法？
	public static InetAddress getByName(String host)：在给定主机名的情况下确定主机的 IP 地址
	public String getHostName()：获取此 IP 地址的主机名
	public String getHostAddress()：返回 IP 地址字符串（以文本表现形式）
~~~
